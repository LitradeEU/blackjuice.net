# BlackJuice.net

Prima preview statica del blog BlackJuice, ispirata al ritmo editoriale di Every ma con identita, contenuti e strumenti creator originali.

## Preview locale

```bash
python3 -m http.server 4173
```

Poi apri `http://127.0.0.1:4173`.

## Cosa contiene

- Homepage pubblica con griglia editoriale, sezioni, newsletter e archivio.
- Pagina articolo navigabile da hash URL.
- Area creator su `#creator` con inserimento pubblicazione, immagine di copertina, metadati, link correlati, bozze/pubblicazione e analytics locali.
- Persistenza in `localStorage`, utile per preview e prototipazione prima del backend.
